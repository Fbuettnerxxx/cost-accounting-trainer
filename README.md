# TUM Cost Accounting Exam Trainer

Interactive exam trainer for TUM Cost Accounting (Prof. Friedl) — 123 questions with step-by-step solutions.

**🔗 Live site: https://fbuettnerxxx.github.io/cost-accounting-trainer/**

## Topics covered

- Cost Classification
- CVP Analysis
- Support Department Allocation
- Process Costing
- Product / Job Costing
- Material Valuation
- Depreciation
- P&L / Income Statements
- Cost Functions
- Activity-Based Costing
- Contribution Margin
- Make-or-Buy
- Imputed Costs

## Usage

Open [`index.html`](index.html) in a browser, or use the [live site](https://fbuettnerxxx.github.io/cost-accounting-trainer/). No build step or dependencies required.
